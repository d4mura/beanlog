# ☕ BeanLog — コーヒー豆のVivino
## 事業計画書

---

## 1. エグゼクティブサマリー

**プロダクト名（仮）:** BeanLog  
**コンセプト:** コーヒー豆のVivino — スキャン・レビュー・発見のプラットフォーム  
**ターゲット:** スペシャルティコーヒー愛好家 × 海外からの日本コーヒー探索者  
**市場規模:** 日本コーヒー市場 約2.9兆円 / スペシャルティコーヒー市場 248億円（拡大中）  

### 解決する課題
- コーヒー豆の口コミが「ストーリーズ→消滅」で蓄積されない
- 産地・農園・精製方法などのストーリーが断片的で検索できない
- 海外の人が日本のロースターを発見する手段がない
- ロースター横断で「今どこでこの豆が買えるか」を知る方法がない

## 2. 市場分析

### 2.1 マーケット概況
| 指標 | 数値 |
|------|------|
| 日本コーヒー市場規模 | 約2.9兆円 |
| スペシャルティコーヒー市場 | 248億円 |
| スペシャルティ比率（輸入量ベース） | 13.4%（2024年） |
| PostCoffee会員数 | 9万人超 |
| PostCoffeeサブスク継続率 | 約98% |

### 2.2 競合マップ

| プレイヤー | ポジション | 弱点 |
|-----------|-----------|------|
| PostCoffee | サブスク＋ECモール（コーヒー版ZOZOTOWN） | 自社販売が主、UGCレビュー弱い |
| コーヒー豆研究所 | SEOアフィリメディア | 情報が広告寄り、ユーザー参加なし |
| マイベスト | 総合比較サイト | コーヒー特化の深さなし |
| 各ロースターEC | 自社通販 | 横断検索不可、発見性ゼロ |

### 2.3 ポジショニング：なぜBeanLogが勝てるか

**Vivinoが成功した理由のコーヒー版適用：**
- ラベルスキャン → 豆パッケージ撮影で特定（画像認識AI）
- UGCレビュー蓄積 → 身内口コミの「ストック化」
- フレーバープロファイル → テイスティングノートのベクトル化でレコメンド
- 産地ストーリー → 農園・精製方法のデータベース化
- 多言語対応 → 海外ユーザーが日本のロースターを発見

## 3. プロダクト戦略

### 3.1 コア機能（MVP）

| 機能 | 説明 | 優先度 |
|------|------|--------|
| 📸 豆パッケージスキャン | カメラで撮影→豆を特定→情報表示 | P0 |
| 📝 テイスティングレビュー | フレーバーホイール選択＋フリーテキスト | P0 |
| 🔍 豆データベース | 産地/品種/精製/焙煎度/ロースターの構造化DB | P0 |
| 🤖 類似豆レコメンド | フレーバーノートのベクトル類似検索 | P0 |
| 🌐 多言語対応（日/英） | 最初から英語対応で海外ユーザー獲得 | P1 |
| 🏪 ロースターページ | ロースター情報＋取扱豆一覧＋地図 | P1 |
| 🛒 購入リンク連携 | ロースターECへの導線 | P1 |

### 3.2 フェーズ別ロードマップ

```
Phase 0: MVP（2ヶ月）
├── Webアプリ（PWA）
├── 豆DB（手動＋スクレイピングで初期データ200件）
├── レビュー投稿・閲覧
├── 基本検索（産地/焙煎度/フレーバー）
└── 日英対応

Phase 1: グロース（3-6ヶ月）
├── 画像認識によるパッケージスキャン
├── フレーバーベクトル化＋レコメンドエンジン
├── ロースター公式ページ機能
├── ネイティブアプリ（React Native）
└── SNSシェア機能

Phase 2: マネタイズ（6-12ヶ月）
├── ロースター向けダッシュボード（有料）
├── アフィリエイト連携
├── プレミアム会員（詳細データ、先行情報）
├── API提供（B2B）
└── アジア展開（台湾/韓国/タイ）
```

### 3.3 技術スタック（推奨）

| レイヤー | 技術 | 理由 |
|---------|------|------|
| Frontend | Next.js + TypeScript | SSR/SEO + PWA対応 |
| Backend | Rust (Axum) or Python (FastAPI) | Rust学習中なら実践の場に / 速度重視ならFastAPI |
| DB | PostgreSQL + pgvector | フレーバーベクトル検索 |
| 画像認識 | CLIP or カスタムモデル | パッケージ→豆特定 |
| レコメンド | sentence-transformers + pgvector | テイスティングノート類似検索 |
| Infra | Vercel (FE) + Railway/Fly.io (BE) | 低コスト＋スケーラブル |
| Auth | Clerk or Supabase Auth | ソーシャルログイン対応 |
